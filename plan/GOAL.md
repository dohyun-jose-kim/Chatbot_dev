# Project Goal

## 핵심 목표 (Must Achieve)

**PubMedBERT로 논문 데이터를 임베딩하고, LLM과 연결한 RAG 기반 ChatBot을 구현한다.**

## 구성 요소

1. **Embedding** — PubMedBERT를 활용한 논문 데이터 임베딩
2. **Vector DB** — 임베딩 벡터 저장 및 검색
3. **Retriever** — 사용자 질문과 관련된 논문 검색
4. **LLM** — 검색된 논문 기반 답변 생성
5. **ChatBot** — 위 파이프라인을 통합한 대화형 인터페이스
